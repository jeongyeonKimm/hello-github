# 맥북 팁
저도 아는게 없고 단축키 정도?
앱 종료: cmd+Q
저장: cmd+S
복붙: cmd+C, cmd+V
새창: cmd+N
화면 캡쳐, 녹화: cmd+shift+3/4/5
강제 종료: option + cmd + esc
프로그램 설치는 homebrew를 이용 한답니다.
분할 화면은 magnet을 이용해요.
